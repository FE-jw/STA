STA Scroll Trigger Animation Ver 1

가볍다! 빠르다! 쉽다! Vanilla JS + CSS 스크롤 트리거 애니메이션 플러그인

Default

요소에 data-sta 속성만 추가하면 기본 효과 적용 끝!
이 외에 dataset 추가 설정으로 손쉽게 인터랙티브한 웹을 구현할 수 있습니다.

<div data-sta></div>

Delay

data-sta-delay 속성으로
애니메이션 시작을 지연시킬 수 있습니다.


<div data-sta
	data-sta-delay="300">
					

<div data-sta
	data-sta-delay="700">
					

<div data-sta
	data-sta-delay="500">

Get Started

  1. Add Markup

    <div data-sta></div>

  2. Add CSS

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/fe-jw/STA/Ver1/sta@1.min.css">

  3. Add JS

    <script src="https://cdn.jsdelivr.net/gh/fe-jw/STA/Ver1/sta@1.min.js"></script>

  4. Initialize

    window.addEventListener('load', function(){
	document.querySelectorAll('[data-sta]').STA();
});
